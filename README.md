## Introduction
高専を卒業し、新卒でNTTに入社。これまで約4年間通信設備の構築や保守 設計業務などを担当した。
大きな会社で働く中で 年功序列や成果が正当に評価されない環境に違和感を感じ、退職を決意する。
しかし、会社に依存していた自分の無力さに気づき、IT社会における自分の市場価値を高めるため、プログラミングを独学する。
今までの知識・経験を活かし　身につけた自走力を武器に 新たな環境で成長し続ける。
